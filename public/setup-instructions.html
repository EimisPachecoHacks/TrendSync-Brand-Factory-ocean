Setup Instructions

1 Open the Application

Navigate to the TrendSync app in your browser (usually http://localhost:5173)

Open App →

2 Click "Create Account"

On the login page, click the "Create Account" tab (right side)

3 Fill in the Form

Enter these exact details:

  • Full Name: Demo User
  • Email: demo@trendsync.ai
  • Password: TrendSync2025!

4 Create & Sign In

Click "Create Account" button, then switch to "Sign In" tab and login with the same credentials.

5 Add Gemini API Key

After logging in:

  • Go to Settings in the sidebar
  • Get your API key from Google AI Studio
  • Paste it and click "Save Settings"

💡 Why is this needed?

The database is empty when you first start the app. The "Invalid login credentials" error means the account doesn't exist yet. Once you create it through the signup form, you can log in normally.
